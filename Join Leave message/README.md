# Custom Join / Leave message - 自定义加入/退出消息

## 插件介绍

一款支持自定义登入、登出显示的插件，非常的简洁，支持高版本！  
还可以支持标题文字显示！

---

## 使用方法

下载此插件，放入 plugin 文件夹，重启服务器并更改配置文件！

---

## 插件命令

`/Joinmessage set <Message>` -> 改变玩家登入信息，使用 **%player%** 替代玩家 ID  
`/Leavemessage set <Message>` -> 改变玩家登出信息，使用 **%player%** 替代玩家 ID  
`/Languages` -> 查看所有支持的语言
